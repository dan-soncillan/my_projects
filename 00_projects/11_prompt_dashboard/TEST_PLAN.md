# テスト計画（MVP）

## 1. 基本動作
- [ ] 送信1回で +1 記録される（ChatGPT）
- [ ] 送信1回で +1 記録される（Claude）
- [ ] 送信1回で +1 記録される（Gemini）

## 2. Enter/Shift+Enter
- [ ] Enter送信で +1
- [ ] Shift+Enter はカウントされない
- [ ] IME変換確定のEnterで誤カウントしない（可能な範囲で）

## 3. デデュープ
- [ ] クリックとEnterが同時発火しても +1 のみ
- [ ] 連打しても2秒以内は1回に丸められる

## 4. 日付境界
- [ ] 23:59 → 00:00 の跨ぎで day が正しく切り替わる

## 5. マルチタブ
- [ ] 複数タブで送信しても正しく加算される
- [ ] タブごとのデデュープが機能する

## 6. ダッシュボード
- [ ] Today合計が正しい
- [ ] 7日推移が正しい
- [ ] ツール別内訳が正しい

## 7. CSV
- [ ] CSV出力が送信イベント単位で整形される
- [ ] 文字化けせずに開ける（UTF-8）

## 8. 削除
- [ ] 全削除でデータが消える
- [ ] 削除後にダッシュボードが0表示になる
