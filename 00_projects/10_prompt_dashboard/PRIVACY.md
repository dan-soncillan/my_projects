# プライバシーポリシー（個人向け / MVP）

## 1. 収集する情報
本拡張は、以下の情報のみをローカルに保存します（送信単位で個別記録）。
- 一意ID（UUID v4、PK）
- タイムスタンプ（YYYY-MM-DD hh:mm:ss形式）
- ツール種別（ChatGPT / Claude / Gemini / Cursor）
- ソース種別（webui / manual）

## 2. 収集しない情報
本拡張は、以下を **取得・保存しません**。
- プロンプト本文（入力した文章）
- 会話内容・返信内容
- アカウント情報（メールアドレス等）
- クレジットカード情報
- Cookie などの認証情報

## 3. データの保存場所
- ブラウザのローカルストレージ（例：chrome.storage.local）
- 外部サーバには送信しません（MVP仕様）

## 4. データの共有・第三者提供
- 第三者への提供は行いません
- 広告目的での利用は行いません

## 5. ユーザーによる削除
- 拡張の設定画面から全データを削除できます
- 拡張をアンインストールすることでローカルデータも削除されます（ブラウザ仕様に依存）

## 6. 変更
- 仕様変更がある場合は `CHANGELOG.md` に記録します
